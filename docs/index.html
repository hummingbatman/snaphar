SnapHAR GitHub

One-click HAR capture.

SnapHAR records a page's network traffic and exports a HAR 1.2 file — without opening DevTools. Start, reproduce the issue, stop. Done.

Download Source Privacy

Coming soon to the Edge Add-ons and Chrome Web Store.

What it does

No DevTools required
Record straight from the toolbar popup, with a live timer and request count.
Preserve-log by default
Capture survives reloads and redirects; each hop is its own entry.
Safe by default
Sensitive headers, cookies, and URL tokens are redacted automatically.
Edge, Chrome & Chromium
One Manifest V3 build. The exported HAR records whichever browser you used.
Valid HAR 1.2
Re-import into the Network panel or any HAR analyzer. Bodies and full timings included.

How it works

  1. Open the popup and click Start Recording.
  2. Reproduce the issue — reload, click, trigger the bug.
  3. Click Stop & Export to save the .har.

Install

# until it's in the stores, load it unpacked:
open  edge://extensions   # or chrome://extensions
enable Developer mode
click  Load unpacked  →  select the folder

Privacy

No servers, no analytics, no telemetry. Everything is assembled locally and saved to your Downloads folder. The debugger permission powers capture and is active only while you record. Details.
